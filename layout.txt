## RESPONSIVE LAYOUT RULES

### Mobile (<768px)
- Single column layout
- Stack everything vertically:
HEADER → MAIN → FOOTER

Inside MAIN:
HERO → RELEASE → SHOWS → BANDA → CONTACTO → MUSICA

---

### Tablet (≥768px)
- HEADER / MAIN / FOOTER stacked
Inside MAIN:
- HERO full width
- RELEASE + SHOWS in 2 columns
- BANDA full width
- CONTACTO full width
- MUSICA full width

---

### Desktop (≥1024px)
Inside MAIN:
- HERO full width
- RELEASE (left) + SHOWS (right)
- BANDA (left) + CONTACTO (right)
- MUSICA full width or grid of cards depending on content

Use CSS Grid as default layout system.
